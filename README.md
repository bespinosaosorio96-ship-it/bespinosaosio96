- 🔭 **Proyectos actuales:** Optimizando arquitecturas en [CultuVivo](https://github.com/luish-gonzalez/Proyecto_CultuVivo_LealShirlyGomezAndresEspinosaBrayanGonzalezLuis) y experimentos de IA.
